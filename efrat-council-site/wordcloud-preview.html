תצוגה מקדימה - ענן מילים (SVG)

SVG version. Words are positioned freely, rotated, and layered. Hover any word to isolate it.
תקציב וכספים תשתיות ובנייה אנשים וקהילה מנהל ותהליך מקומות / שכונות
What's different in SVG:
  • True hover isolation — when you mouse over the cloud, every word fades except the one under your cursor. This works only with SVG; CSS-flex doesn't get you the same focus effect cleanly.
  • Words tucked into gaps instead of grid-flow positioning — "ביטחון" and "שמירה" rotate 90°, sitting in the left margin. "מורדות הזית" and "תיירות" tilt slightly. The cloud feels less like a list and more like a constellation.
  • Crisp at every size — SVG scales, so this will look identical on phones, retina monitors, and a projector. The CSS version stretches but uses raster-rendered text.
  • One-click navigation is easy to add: every <text> can become a link to topics.html?q=<word>. The handler is two lines of JS that reads the click target's data-word attribute.
  • The viewBox does the responsive work — it's set to 900 × 540 with preserveAspectRatio="xMidYMid meet", so positions stay fixed relative to each other while the whole thing scales to fit the page width.

If this version reads well, I'll wire it into the live topics page above the search bar, with click-to-search hooked up. Word positions and sizes are pure data — easy to tweak any single one without touching the others.
